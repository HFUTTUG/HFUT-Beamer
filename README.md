# 合肥工业大学 Beamer 模板集合

本项目是在前人的基础上编写的展示用 Beamer 模板集合。目前提供以下几种模板：

| 模板名称             | 原作者/发布者 |
| ---------------- | ------ |
| cambridgeUS-hfut | 贾宇航    |
| hfut             | [孙晓](https://github.com/sxhfut/)|

可以在`example.tex`中的`\usetheme{...}`中进行调整主题的调整。
